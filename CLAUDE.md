# CLAUDE.md

The AkaLab-Tech claude-plugins marketplace repository hosts a Claude Code plugin marketplace registry (`.claude-plugin/marketplace.json`). It catalogs plugins including atelier, claude-roadmap-tools, and integration plugins for Coolify, Neon, and Vercel. The repository is the source of truth for plugin discovery and distribution within the AkaLab-Tech Claude Code ecosystem.
